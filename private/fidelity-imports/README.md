# Fidelity Import Drop Zone

Drop Fidelity portfolio exports here when you want Codex to refresh Flory Freedom.

Preferred files:
- Fidelity account positions export as `.csv` or `.xlsx`
- Detailed holdings export as `.xls` or `.xlsx`
- Statement PDFs only when no spreadsheet export is available

Suggested task prompt:

```text
Update Flory Freedom from the newest Fidelity export in private/fidelity-imports.
Refresh portfolio totals, cash, allocation, concentration, international exposure, and any changed account notes.
Validate the app and deploy it.
```

Files in this folder are ignored by Git so account statements and exports do not get pushed to GitHub.
